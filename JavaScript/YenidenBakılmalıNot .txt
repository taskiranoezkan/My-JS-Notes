Rest API Nedir?
REST API, geliştiricilerin HTTP protokolünü kullanarak GET ve POST gibi isteklerde bulunup, bu isteklere çeşitli formatlarda yanıt aldığı bir dağıtık sistemdir.
REST (REpresentatZonal State Transfer), Temsili Durum Aktarımı olarak Türkçede de kullanılır.
REST’in tüm prensiplerini yerine getiren API’ler ise RESTful olarak nitelendirilir.

JavaScript Prompt Nedir?
JavaScript'te kullanıcıdan veri almak iki farklı şekilde yapılır.Birisi prompt komutuyla diğeri ise form yollu ile bilgi alınır.
Form HTML üzerinde yer almsı, Prompt komutunda ekrana küçük diyalog pencere açılması olarak bilinir.
Ex. let değisken=window.prompt('Mesaj','Varsayılan Değer')

Kullanıcı 'Tamam' a tıklarsa giriş değeri döndürülür,Kullanıcı giriş yapmdan 'Tamam' a tıklarsa boş string döner.
giriş yapmış olsa dahi 'Cancel' a tıklarsa null döner
